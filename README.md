## GitHub-structuur

Gebruik deze structuur in je repository:

```text
repo/
├─ README.md
└─ images/
   ├─ scenario-1.png
   ├─ scenario-2.png
   ├─ scenario-3.png
   └─ scenario-4.png
```

Upload `README.md` samen met de map `images/`, anders worden de afbeeldingen niet geladen.
